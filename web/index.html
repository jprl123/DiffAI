Guia de uso

Compare two document versions and get a standardized redline PDF with a change summary.

For a guided walkthrough of the main buttons, open Settings → Replay tutorial.

Single compare

  1. Drop (or click to choose) the Base document — the original.
  2. Drop the Revised document — the updated version.
  3. Wrong order? Use the swap button between the two areas.
  4. Adjust output options and click Compare. Extras never block the main redline.

Batch (folders)

  1. Drop folders or click to open the file explorer.
  2. Pick the folder with originals and the folder with revised files.
  3. Pairs are matched by file name or content (no AI).
  4. Click View pairs, then Process.

History

Every comparison stays in History, even after you close the app.

Compare duas versões de um documento e receba um PDF redline padronizado, com resumo das mudanças.

Comparar (arquivo único)

  1. Arraste (ou clique para escolher) o Documento base — a versão original.
  2. Arraste o Documento revisado — a versão atualizada.
  3. Selecionou na ordem errada? Use o botão de inverter entre as duas áreas.
  4. Ajuste as opções de saída e clique em Comparar.

Lote (pastas)

  1. Arraste as pastas ou clique para abrir o explorador de arquivos (Finder no Mac).
  2. Selecione a pasta com os documentos originais e a pasta com os revisados.
  3. O sistema encontra os pares pelo nome dos arquivos (variações como “v2”, “_rev” e “(1)” são ignoradas) ou pelo conteúdo.
  4. Clique em Ver pares para conferir a prévia do pareamento e depois em Processar: cada par gera seu próprio redline. Se um par falhar, os demais continuam.

Histórico

A aba Histórico guarda todas as comparações já feitas, mesmo depois de fechar o app. Filtre por status, busque pelo nome do arquivo e reabra qualquer resultado.

Marcações do redline

texto inserido azul sublinhado — conteúdo novo no documento revisado
texto excluído vermelho tachado — conteúdo removido do documento base
modificação âmbar — trecho presente nos dois, com alterações
movimentação verde — mesmo conteúdo em outra posição

Sinal vs. ruído (categorias)

Conteúdo é a mudança substantiva — o que merece a sua atenção. Rotineiras são alterações de data, versão, numeração de página, espaçamento ou pontuação: contadas e marcadas, mas separadas para não competir com o que importa. Formatação é o mesmo texto com visual diferente (negrito, itálico etc.). Tabelas e Imagens têm categorias próprias. Use os filtros do painel “Ver mudanças” para focar só no substantivo.
